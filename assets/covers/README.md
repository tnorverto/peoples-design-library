# Cover images

Drop card cover images in this folder (jpg/png/webp).

- Use simple file names: lowercase, no spaces — e.g. `studio-mk27.jpg`
- On the site (signed in as admin), click the 🖼 button on a card and paste:
  `assets/covers/your-file-name.jpg`
- The image goes live the next time the site is deployed (push to main).

Good size: roughly 680×420 or similar landscape crop — that's what the hover
preview shows. Bigger is fine; it gets scaled down.
